Perfect! I can see the issue. Your font styling problems are likely due to missing Google Fonts imports and font family configurations. Here's what you need to transfer to your other React.js project:

1. Google Fonts Import
Add this to your HTML <head> section:

<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;500;600;700&family=Raleway:wght@300;400;500;600&display=swap" rel="stylesheet">
2. CSS Variables and Font Configuration
Add these CSS variables to your main CSS file:

:root {
  /* Font families */
  --font-display: 'Playfair Display', serif;
  --font-body: 'Raleway', sans-serif;
  
  /* Custom colors */
  --ocean-blue: #0F4C81;
  --tropical-green: #2E8B57;
  --golden-sand: #D4AF37;
  --soft-ivory: #F8F5F0;
  --rich-brown: #3D2C1F;
  --text-dark: #333333;
  --cream: #FFFAF0;
  
  /* Base ShadCN variables */
  --primary: #0f766e;
  --primary-foreground: #ffffff;
  
  /* Luxury theme variables */
  --secondary: #D4AF37; /* Gold accent */
  --secondary-foreground: #000000;
  --accent: #103556; /* Dark blue */
  --accent-foreground: #ffffff;
  
  /* Commonly used colors */
  --gold: #D4AF37;
  --gold-light: #e9d58a;
  --gold-dark: #aa8c27;
  /* Gradients and backgrounds */
  --luxury-gradient: linear-gradient(to right, var(--accent), var(--accent-dark, #0a2541));
  --cream-bg: #f8f7f2;
}
/* Global luxury styling */
h1, h2, h3, h4, h5, h6 {
  font-family: 'Playfair Display', serif;
  color: var(--accent);
}
body {
  font-family: 'Raleway', sans-serif;
}
3. Tailwind Config Font Family
If you're using Tailwind CSS, update your tailwind.config.js:

module.exports = {
  theme: {
    extend: {
      fontFamily: {
        'display': ['Playfair Display', 'serif'],
        'sans': ['Raleway', 'sans-serif'],
      }
    }
  }
}
4. Component Usage
In your components, use the fonts like this:

// For headings
<h1 className="font-['Playfair_Display'] text-4xl font-bold">
  Title Text
</h1>
// For body text (default)
<p className="text-lg">
  Body text will automatically use Raleway
</p>

Missing Google Fonts link in your HTML
Missing CSS font-family declarations
Incorrect Tailwind font class usage
